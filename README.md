# 🧠 English Vocabulary - Memory Game

¡Bienvenido/a! Juego de memoria clásico diseñado para aprender vocabulario en inglés de forma interactiva.

**[Link al proyecto en vivo](https://taticode.github.io/memory-game/)**

## 🚀 Características
- **Aprendizaje Visual:** Asociación palabra-imagen.
- **Responsive Design:** Optimizado para móvil y escritorio.
- **Lógica JavaScript:** Algoritmos de barajado (Shuffle) y manejo de estados.

## 🛠️ Tecnologías
* **HTML5 / CSS3 (Sass & BEM)**
* **JavaScript (Vanilla)** - Manipulación del DOM y eventos.
